Consignes données dans le bloc-notes "Script-Console" de Capytale V2 pour l'activité d'analyse de données sur les fichiers INSEE de prénoms par département

Description des colonnes des fichiers joints:

  1. sexe: 1 pour garçon, 2 pour fille
  2. preusuel: prénom tel qu'il est utilisé (sauf pour "_PRENOMS_RARES"
  3. annais: année considérée
  4. dpt: numéro du département considéré
  5. nombre: nombre de fois où le prénom a été donné

------------------------------------------------------------------------------------------------------------------------------------------

CONSEIL: les fichiers de l'Ile-de-France, sur lesquels portent les questions, sont assez lourds et donc assez longs à charger. Pour gagner du temps pour vos tests je vous conseille d'utiliser le fichier Dept48 (Lozère), qui est très nettement plus petit.

Problème D: Faites tourner le code ci-dessous pour voir ce qu'il fait, puis tracez la courbe de l'évolution de l'attribution du prénom réponse à la question précédente entre 2000 et 2022.

import matplotlib.pyplot as plt

# Données
x = [1, 2, 3, 4, 5] # Axe des abscisses
y = [2, 3, 5, 7, 11] # Axe des ordonnées


# Création de la courbe
plt.plot(x, y)


# Ajout d'un titre et des labels pour les axes
plt.title("Exemple de Courbe")
plt.xlabel("Axe des X")
plt.ylabel("Axe des Y")


# Affichage de la courbe
plt.show()
